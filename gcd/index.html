Brute force List


Pseudocode

  1.   Get three numbers (n1,n2,n3)

  2.   Take n1 n2 to a while loop with a variable i

  3.   If n1 and n2 mod i is zero, the i is gcd value

  4.   Else i++

  5.   Get n1 and n2's gcd value

  6.   Let n2 be n1 and n2's gcd value

  7.   And take n2 n3 to a while loop with a variable i

  8.   Take n2 n3 to a while loop with a variable i

  9.   If n2 and n3 mod i is zero, the i is gcd value

  10.   Else i++

  11.   Get the three number's gcd value

Euclidean algorithm List


Pseudocode

  1.   Get three numbers (n1,n2,n3)

  2.   Make their gcd likes

  3.   $gcd(n1,n2,n3)=gcd( gcd(n1,n2),n3 )$

  4.   if b != 0 do the while

  5.   if "a = a mod b , b = b mod a" ,then the while loop keep going.

  6.   after the loop ending,a+b is the two numbers' gcd value

the successive subtraction List


Pseudocode

  1.   Get three numbers (n1,n2,n3)

  2.   Make their gcd likes

  3.   $gcd(n1,n2,n3)=gcd( gcd(n1,n2),n3 )$

  4.   when $m != n$ doing the while loop

    1.   when $m > n$ ;Let $m = m - n$

    2.   when $m < n$ ;Let $n = n - m$

  5.   when ending the loop m is the gcd value
